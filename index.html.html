Physical Quantities and Measurement
  • Physical quantities → characteristic or property of an object or phenomenon that can be measured
  • Quantifiable property ascribed to a particular phenomenon, body, or substance
  • Any number used to describe an observation of a physical phenomenon quantitatively
  • Grouped into two: basic and derived

Basic Quantities

  • Basic quantities → building blocks for formulation of derived physical quantities
  • More fundamental than others; defined only by the procedure used to measure them
  • Units in which they are measured → called fundamental units
  • The fundamental (basic) physical quantities are:
    • Mass → kilogram (kg) [base unit]
    • Length → meter (m) [base unit]
    • Time → second (s) [base unit]
    • Electric current → ampere (A)
    • Temperature → kelvin (K)
    • Amount of substance → mole (mol)
    • Luminous intensity → candela (cd)

Prefixes

  • Once fundamental units are defined → introduce larger and smaller units for the same physical quantities
  • Names of additional units derived by adding a prefix to the name of the fundamental unit
  • Examples: kilo = \(10^3\), milli = \(10^{-3}\), micro = \(10^{-6}\), nano = \(10^{-9}\)
\(1 \, \text{kilometer} = 1 \, \text{km} = 10^3 \, \text{meters} = 10^3 \, \text{m}\)
\(1 \, \text{kilogram} = 1 \, \text{kg} = 10^3 \, \text{grams} = 10^3 \, \text{g}\)
\(1 \, \text{kilojoule} = 1 \, \text{kJ} = 10^3 \, \text{Joules} = 10^3 \, \text{J}\)
\(1 \, \text{nanometer} = 1 \, \text{nm} = 10^{-9} \, \text{m}\)   (very small scale)
\(1 \, \text{micrometer} = 1 \, \mu\text{m} = 10^{-6} \, \text{m}\)
\(1 \, \text{millimeter} = 1 \, \text{mm} = 10^{-3} \, \text{m}\)

★ Example: Consider the speed of a car to be ninety kilometers per hour (90 km/h). What is this speed in m/s?
Solution:
Given \(v = 90 \, \text{km/h}\), required \(v\) in \(\text{m/s}\).

\(1 \, \text{kilometer} = 1 \, \text{km} = 10^3 \, \text{meters} = 10^3 \, \text{m}\)
\(1 \, \text{hour} = 60 \, \text{minutes} = 1 \, \text{h} \times 60 \, \text{minutes/h} \times 60 \, \text{s/minute} = 3600 \, \text{s}\)

\(90 \, \text{km/h} = \dfrac{90 \, \text{km} \times 10^3 \, \text{m/km}}{1 \, \text{h} \times 60 \, \text{minutes/h} \times 60 \, \text{s/minute}}\)

\(= \dfrac{90 \times 10^3 \, \text{m}}{3600 \, \text{s}} = 25 \, \text{m/s}\)   (answer)

Derived Quantities

  • Derived quantities → physical quantities derived from the basic ones
  • Formulated from the combination of two or more basic physical quantities
  • This category includes:
    • Velocity in \(\text{m/s}\) → derived from distance and time
    • Volume in \(\text{m}^3\) → derived from three lengths
    • Force in \(\text{N} = \text{kg} \cdot \text{m/s}^2\) → derived from mass, length, and time
  • What is better to remember derived quantities → keep all the seven basic quantities because all the rest are derived ones

Scalars and Vectors

  • Physical quantities can also be grouped into two: scalars and vectors
  • Scalar: has magnitude only (e.g., mass, temperature, speed, distance)
  • Vector: has magnitude and direction (e.g., velocity, force, displacement)

Measurement and Uncertainty
Accuracy, Precision, and Error Analysis
  • Measurement → act of comparing a physical quantity with a reference or standard
  • All measurements have uncertainties associated with them
  • Accuracy depends on:
    • Sensitivity of the apparatus
    • Skill of the person carrying out the measurement
    • Number of times the measurement is repeated
  • Errors cannot be avoided but can be minimized
  • Error → deviation of measurement outcome from the true or average value
  • Uncertainty is also called error

Accuracy and Precision

  • Scientists make repeated measurements to ensure precision and accuracy
  • Precise → measurements yield very similar results when repeated in the same manner
  • Accurate → measurement yields a result very close to the true or accepted value
  • Precise values agree with each other; accurate values agree with a true value
  • Accuracy → how close the measured value approaches the true or expected value
  • Precision → measured outcomes are close to each other or less scattered
  • Uncertainty → quantitative measure of how much measured values deviate from a standard or expected value
Error formula:
\(\text{Error} = \text{measured value} - \text{true value}\)
  • Accuracy of a measured value is indicated by:
    • Using \(\pm\) symbol with uncertainty value
    • Fractional error
    • Percent error
Percent error formula:
\(\% \text{error} = \dfrac{\text{measured value} - \text{true value}}{\text{true value}} \times 100\%\)
★ Example 1:
Suppose that the measured value of the acceleration of gravity is found to be \(9.44 \, \text{m/s}^2\). The accepted value is \(9.80 \, \text{m/s}^2\). Calculate the percentage error.

Solution:
Difference = \(9.80 \, \text{m/s}^2 - 9.44 \, \text{m/s}^2 = 0.36 \, \text{m/s}^2\)
\(\% \text{error} = \dfrac{0.36 \, \text{m/s}^2}{9.80 \, \text{m/s}^2} \times 100\% = 3.7\%\)
  • Precision → how close the agreement is between repeated measurements
  • An accurate measurement is precise, but precise measurement does NOT mean accurate
  • Error can be negative (below true value) or positive (above true value)

Types of Errors

Systematic Error

  • Results from defect in the measuring device itself
  • Makes measurement outcomes consistently too high or too low
  • Minimized by:
    • Calibrating the device using standards
    • Replacing defective device with calibrated ones

Random Error

  • Caused by fineness of scale division of a measuring device
  • All measurement outcomes are equally probable
  • Outcomes randomly scattered around the true value
  • More complex and hard to handle
  • Minimized by multiple or repeated measurements and taking the average

Significant Figures
Rules and Basic Operations
  • Significant figures (digits) → number of meaningful digits used to describe a measurement outcome
  • Digits in any measurement that are known with certainty plus one digit that is uncertain (last digit)
  • The number of meaningful digits implies the error in the measurement
  • More decimal places in a significant digit → more accurate the measurement outcome

Standard Rules for Significant Figures

Rule 1

  • All non-zero numbers are significant
  • Examples: 3.1428 (5), 23 (2), 3.14 (3), 469 (3)

Rule 2

  • Zeros between numbers are always significant
  • Examples: 7.053 (4), 7053 (4), 302 (3)

Rule 3

  • Zeros to the left of the first nonzero digit are NOT significant
  • Examples: 0.0056 (2), 0.0789 (3), 0.00098 (2), 0.98 (2)

Rule 4

  • In numbers with digits to the right of a decimal point, zeros to the right of the last nonzero digit are significant
  • Examples: 43 (2), 43.0 (3), 43.00 (4), 0.00200 (3), 0.40050 (5)

Rule 5

  • In numbers with no decimal point ending in zeros → zeros are ambiguous
  • Confusion avoided using scientific notation
  • Examples: \(3.6 \times 10^3\) (2), \(3.60 \times 10^3\) (3), \(3.600 \times 10^3\) (4)
Additional examples:
\(0.0009\) (1), \(6 \times 10^2\) (1), \(2\) (1), \(0.4\) (1), \(10\) (1)
\(11\) (2), \(2.0\) (2), \(0.30\) (2), \(1.0 \times 10\) (2), \(0.025\) (2)
\(234\) (3), \(201\) (3), \(2.00\) (3), \(0.102\) (3), \(4.00\) (3), \(0.200\) (3)

Basic Operations with Significant Figures

  • Addition and Subtraction:
    • Number of significant figures in result → same as smallest number of significant digits in sum/difference
    • Number of decimal places in result → should equal smallest number of decimal places of any term
★ Example 1 (Addition/Subtraction):
Calculate \(9.65 \, \text{cm} + 8.4 \, \text{cm} - 2.89 \, \text{cm}\).

Step 1: Perform the calculation:
\(9.65 + 8.4 - 2.89 = 15.16 \, \text{cm}\)
Step 2: Identify the least precise measure → 8.4 cm (1 decimal place)
Step 3: Round the result to 1 decimal place
Answer: \(15.2 \, \text{cm}\)
Example: \(34.6 + 17.8 + 15 = 67\) (no decimal places)
  • Multiplication and Division:
    • Number of significant figures in product/quotient → same as least accurate factor (lowest number of significant figures)
★ Example 2 (Multiplication):
Multiply 8.71 cm by 3.2 cm.

Step 1: Perform the multiplication:
\((8.71 \, \text{cm})(3.2 \, \text{cm}) = 27.872 \, \text{cm}^2\)
Step 2: Identify least significant factor → 3.2 has 2 significant figures
Step 3: Round the result to 2 significant figures
Answer: \(28 \, \text{cm}^2\)
★ Example 3 (Division):
Calculate \(\frac{42 \, \text{N}}{3.22 \, \text{m} \times 2.005 \, \text{m}}\).

Step 1: Calculate the denominator:
\(3.22 \times 2.005 = 6.4561 \, \text{m}^2\)
Step 2: Perform the division:
\(\frac{42}{6.4561} = 6.97015 \, \text{N/m}^2\)
Step 3: Identify least significant factor → 42 has 2 significant figures
Step 4: Round the result to 2 significant figures
Answer: \(7.0 \, \text{N/m}^2\)
★ Example 4 (Multiplication):
Calculate \(3.1416 \times 2.34 \times 0.58\).

Step 1: Perform the multiplication:
\(3.1416 \times 2.34 = 7.351344\)
\(7.351344 \times 0.58 = 4.26377952\)
Step 2: Identify least significant factor → 0.58 has 2 significant figures
Step 3: Round the result to 2 significant figures
Answer: \(4.3\)
★ Example 5 (Division):
Calculate \(\frac{3840}{285.3}\).

Step 1: Perform the division:
\(\frac{3840}{285.3} = 13.459516\)
Step 2: Identify least significant factor → 3840 has 3 significant figures
Step 3: Round the result to 3 significant figures
Answer: \(13.5\)
★ Example 6 (Division with decimals):
Calculate \(\frac{360.0}{3.000}\).

Step 1: Perform the division:
\(\frac{360.0}{3.000} = 12\)
Step 2: Identify significant figures in inputs → both have 4 significant figures
Step 3: Report result with 4 significant figures
Answer: \(12.00\)

Scalars and Vectors
Definitions, Representations, and Basic Properties

Scalar Quantities

  • Defined by a number and a unit only
  • Obey the rules of ordinary algebra
  • Negative of a scalar has no meaning
  • Examples: distance, speed, power, energy, current, temperature, mass, time, area, volume, pressure, density

Vector Quantities

  • Defined by both magnitude and direction
  • Obey the law of vector algebra
  • Negative of a vector → direction is changed
  • Examples: force, torque, velocity, displacement, acceleration, linear momentum, angular momentum, electric field, gravitational field, magnetic field
  • Magnitude of vector \(\vec{A}\) → written as \(|\vec{A}|\) or \(A\) → always a positive number
  • Direction of a vector → described by an angle (\(\theta\)) or unit vectors (\(\hat{i}, \hat{j}, \hat{k}\) or \(\hat{x}, \hat{y}, \hat{z}\))

Representation of Vectors

  • Vectors represented by:
    • Bold face letters → A, B, C
    • Letter with an overhead arrow → \(\vec{a}, \vec{v}, \vec{p}, \vec{\tau}\)
    • Geometrically → directed line segment called an arrow (→)
  • Parts of an arrow:
    • Tail → initial point
    • Head (tip) → shows the direction of the vector
  • Magnitude of the arrow → proportional to the magnitude of the vector
★ Example: Given vectors \(\vec{a}_1 = 3 \, \text{m/s}^2\), \(\vec{a}_2 = -3 \, \text{m/s}^2\), \(\vec{a}_3 = 2 \, \text{m/s}^2\), \(\vec{a}_4 = 3 \, \text{m/s}^2\), \(\vec{F}_1 = 3 \, \text{N}\), \(\vec{F}_2 = -3 \, \text{N}\), \(\vec{F}_3 = 2 \, \text{N}\).
Only \(\vec{a}_1\) and \(\vec{a}_4\) are equal vectors.

Addition and Subtraction of Vectors
Geometric Methods: Triangular, Parallelogram, and Polygon

Addition of Vectors

  • Combination of two or more vectors to get a single new vector
  • New vector → called Resultant Vector (R)
  • Resultant has the same effect on a body as the two or more vectors actually acting on that body
  • Resultant starts at the beginning of the first vector and ends at the end of the last one
  • Two ways to add vectors: Geometrically or Analytically

Geometric Addition of Vectors

  • Graphical way of adding vectors
  • Must connect vectors head to tail
  • Three methods:
    • Triangular method (2 vectors)
    • Parallelogram method (2 vectors with an angle)
    • Polygon method (3 or more vectors)

Triangular Method

  • Used to add two vectors
  • Resultant forms the third side of the triangle
  • Steps:
    • Connect the vectors head to tail
    • Draw resultant from tail of first vector to head of second vector
\(\vec{R} = \vec{A} + \vec{B}\)
Fig 1: Triangular_method method
[triangular_method.png]

Parallelogram Method

  • Used to add two vectors with a given angle between them
  • Vectors connected tail to tail
  • Draw a parallelogram with the two vectors as sides
  • Major diagonal → sum (resultant)
  • Minor diagonal → difference
Fig 2: Parallelogram method → major diagonal shows resultant \(\vec{R} = \vec{A} + \vec{B}\)
[parallelogram_method.png]
  • Steps:
    • Draw each vector over the head of the other
    • Draw resultant from common tail to common head
  • From this: \(\vec{A} + \vec{B}^* = \vec{B} + \vec{A}^*\) → \(\vec{A} + \vec{B} = \vec{B} + \vec{A}\)
  • This is called Commutative Property of vector addition

Polygon Method

  • Used to add three or more vectors graphically
  • Steps:
    • Connect all vectors head to tail in order
    • Draw resultant from tail of first vector to head of last vector
  • Resultant forms the last side of the polygon
Fig 3: Polygon method → resultant from tail of first to head of last
[polygon_method.png]
  • Using polygon method: \((\vec{A} + \vec{B}) + \vec{C} = \vec{A} + (\vec{B} + \vec{C})\)
  • This is called Associative Property of vector addition

Subtraction of Vectors

  • Difference between two vectors \(\vec{A}\) and \(\vec{B}\) → denoted \(\vec{A} - \vec{B}\)
  • Equivalent to adding \(\vec{A}\) to the opposite of \(\vec{B}\): \(\vec{A} + (-\vec{B})\)
Fig 4: Vector subtraction → \(\vec{A} - \vec{B} = \vec{A} + (-\vec{B})\)
[vector_subtraction.png]

Analytic Addition of Vectors
Parallel and Anti-parallel Vectors
  • Analytical method involves only the usual rules of algebra
  • Avoids the cumbersome and limited precision of graphical addition

Addition of Parallel and Anti-parallel Vectors

  • Parallel vectors:
    • Same direction → angle between them is 0°
    • Resultant vector = sum of magnitudes
    • Direction = same as the original vectors
  • Anti-parallel vectors:
    • Opposite direction → angle between them is 180°
    • Resultant = difference of magnitudes
    • Direction = direction of the greater vector
★ Example 1 (Parallel vectors):
A boy pulls a wood block with \(40 \, \text{N}\) and his sister helps with \(30 \, \text{N}\) in the same direction. What is the resultant force?

Step 1: Identify given values
\(\vec{F}_{\text{boy}} = 40 \, \text{N}\), \(\vec{F}_{\text{girl}} = 30 \, \text{N}\) (same direction)
Step 2: Add the magnitudes since they are parallel
\(\vec{F}_{\text{total}} = \vec{F}_{\text{boy}} + \vec{F}_{\text{girl}} = 40 \, \text{N} + 30 \, \text{N}\)
Step 3: Calculate
\(\vec{F}_{\text{total}} = 70 \, \text{N}\) in the same direction
Answer: \(70 \, \text{N}\)
★ Example 2 (Anti-parallel vectors):
Two forces \(\vec{F}_1 = 20 \, \text{N}\) and \(\vec{F}_2 = 5 \, \text{N}\) act in opposite directions. Find the resultant force.

Step 1: Identify given values
\(\vec{F}_1 = 20 \, \text{N}\) (one direction), \(\vec{F}_2 = 5 \, \text{N}\) (opposite direction)
Step 2: Subtract the magnitudes since they are anti-parallel
\(\vec{F}_{\text{net}} = \vec{F}_1 - \vec{F}_2 = 20 \, \text{N} - 5 \, \text{N}\)
Step 3: Calculate
\(\vec{F}_{\text{net}} = 15 \, \text{N}\) in the direction of the greater force (\(\vec{F}_1\))
Answer: \(15 \, \text{N}\) in the direction of \(20 \, \text{N}\) force

Resolution of a Vector
Rectangular Components and Analytical Addition
  • To add non-parallel and non-perpendicular vectors analytically → need to have the components of the vectors to be added
  • Resolution → technique used to find components
  • Resolution → process of expressing a vector into its rectangular components (nearly always perpendicular to one another) using trigonometric relations
  • Simply put: Resolution of a vector → split a vector into its horizontal and vertical components using trigonometric relations
  • Using components → add or subtract vectors by adding or subtracting the respective components one from the other

Trigonometric Relations

From sine function:
\(\sin \theta = \frac{\text{opposite}}{\text{hypotenuse}} = \frac{R_y}{R}\)

\(R_y = R \sin \theta\) → called the vertical or y-component of vector \(\vec{R}\)
From cosine function:
\(\cos \theta = \frac{\text{adjacent}}{\text{hypotenuse}} = \frac{R_x}{R}\)

\(R_x = R \cos \theta\) → called the horizontal or x-component of vector \(\vec{R}\)

Resultant from Components

  • For three vectors \(\vec{A}\), \(\vec{B}\), and \(\vec{C}\) → magnitude of resultant \(\vec{R}\) is:
\(R = \sqrt{R_x^2 + R_y^2}\)
Fig 6: Resolution of vector R into horizontal (R_x) and vertical (R_y) components
[vector_resolution.png]
  • Where:
\(R_x = A_x + B_x + C_x\)
\(R_y = A_y + B_y + C_y\)
  • \(A_x, B_x, C_x, A_y, B_y, C_y\) → components of the three vectors obtained by resolution
  • Direction \(\theta\) the resultant makes with x-axis → calculated as:
\(\theta = \tan^{-1} \left( \frac{R_y}{R_x} \right)\)

Vector Addition Example
Analytical Method Using Components
★ Example 1: The three displacement vectors in the figure have magnitudes \(a = 2\,\text{m}\), \(b = 6\,\text{m}\), and \(c = 10\,\text{m}\) and angle \(\theta = 30^\circ\).

a) What is the magnitude of resultant vector \(\overrightarrow{R} = \overrightarrow{a} + \overrightarrow{b} + \overrightarrow{c}\)?
b) Find the direction of \(\overrightarrow{R}\).
Fig 7: Vectors a, b, c with given magnitudes and angle θ = 30°
[three_vectors.png]

Solution (a) — Magnitude of Resultant

  • Vector \(\vec{a}\) is along x-axis → only x-component:
\(\vec{a}_x = 2\,\text{m},\quad \vec{a}_y = 0\)
  • Vector \(\vec{b}\) makes angle \(\theta = 30^\circ\) with x-axis → resolve:
\(\vec{b}_x = b \cos 30^\circ = 6\,\text{m} \times 0.866 = 5.2\,\text{m}\)
\(\vec{b}_y = b \sin 30^\circ = 6\,\text{m} \times 0.5 = 3\,\text{m}\)
  • To find components of \(\vec{c}\) → let angle \(\vec{c}\) makes with negative x-axis be \(\beta\)
  • \(\vec{b}\) and \(\vec{c}\) are perpendicular → \(\beta + \theta + 90^\circ = 180^\circ\) (angle sum theorem for a straight angle)
\(\beta + 120^\circ = 180^\circ\)
\(\beta = 60^\circ\)
  • Resolve \(\vec{c}\) (negative x-direction → minus sign for x-component):
\(\vec{c}_x = -c \cos 60^\circ = -10\,\text{m} \times 0.5 = -5\,\text{m}\)
\(\vec{c}_y = c \sin 60^\circ = 10\,\text{m} \times 0.866 = 8.66\,\text{m}\)
  • Sum of x-components:
\(R_x = a_x + b_x + c_x = 2\,\text{m} + 5.2\,\text{m} - 5\,\text{m} = 2.2\,\text{m}\)
  • Sum of y-components:
\(R_y = a_y + b_y + c_y = 0 + 3\,\text{m} + 8.66\,\text{m} = 11.66\,\text{m}\)
  • Magnitude of resultant vector \(\vec{R}\) → Pythagorean theorem:
\(R = \sqrt{R_x^2 + R_y^2} = \sqrt{(11.66\,\text{m})^2 + (2.2\,\text{m})^2}\)
\(R = \sqrt{135.9556 + 4.84} = \sqrt{140.7956} = 11.9\,\text{m}\)

Solution (b) — Direction of Resultant

  • Direction \(\theta\) calculated from trigonometric relations:
\(\tan \theta = \frac{R_y}{R_x} = \frac{11.66\,\text{m}}{2.2\,\text{m}} = 5.3\)
\(\theta = \tan^{-1}(5.3) = 79.3^\circ\)
Answer: The resultant vector is \(\vec{R} = 11.9\,\text{m}\) at \(79.3^\circ\) above the x-axis.

★ Example 2: Three force vectors have magnitudes \(\vec{A} = 100\,\text{N}\), \(\vec{B} = 80\,\text{N}\), and \(\vec{C} = 40\,\text{N}\) as shown below.

a) What is the magnitude of resultant vector \(\vec{R} = \vec{A} + \vec{B} + \vec{C}\)?
b) Find the direction of \(\vec{R}\).
Fig 8: Force vectors A (30° above +x), B (60° above -x), C (53° below -x)
[three_force_vectors.png]
  • Given: \(\vec{A}\) is \(30^\circ\) above x-axis, \(\vec{B}\) makes \(30^\circ\) with y-axis (or \(60^\circ\) above negative x-axis), \(\vec{C}\) is \(53^\circ\) below negative x-axis
  • From the figure → both \(\vec{B}\) and \(\vec{C}\) have negative x-components and \(\vec{C}\) has negative y-component

Solution (a) — Magnitude of Resultant

  • Resolve \(\vec{A}\) (30° above +x-axis):
\(A_x = A \cos 30^\circ = 100\,\text{N} \times 0.866 = 86.6\,\text{N}\)
\(A_y = A \sin 30^\circ = 100\,\text{N} \times 0.5 = 50\,\text{N}\)
  • Resolve \(\vec{B}\) (60° above negative x-axis → negative x, positive y):
\(B_x = -B \sin 30^\circ = -80\,\text{N} \times 0.5 = -40\,\text{N}\)
\(B_y = B \cos 30^\circ = 80\,\text{N} \times 0.866 = 69.28\,\text{N}\)
  • Resolve \(\vec{C}\) (53° below negative x-axis → negative x, negative y):
\(C_x = -C \sin 53^\circ = -40\,\text{N} \times 0.8 = -32\,\text{N}\)
\(C_y = -C \cos 53^\circ = -40\,\text{N} \times 0.6 = -24\,\text{N}\)
  • Sum of x-components:
\(R_x = A_x + B_x + C_x = 86.6\,\text{N} - 40\,\text{N} - 32\,\text{N} = 14.6\,\text{N}\)
  • Sum of y-components:
\(R_y = A_y + B_y + C_y = 50\,\text{N} + 69.28\,\text{N} - 24\,\text{N} = 95.28\,\text{N}\)
  • Magnitude of resultant:
\(R = \sqrt{R_x^2 + R_y^2} = \sqrt{(14.6\,\text{N})^2 + (95.28\,\text{N})^2}\)
\(R = \sqrt{213.16 + 9078.2784} = \sqrt{9291.4384} = 96.39\,\text{N}\)
[three_force_vectors.png]

Solution (b) — Direction of Resultant

\(\tan \theta = \frac{R_y}{R_x} = \frac{95.28\,\text{N}}{14.6\,\text{N}} = 6.53\)
\(\theta = \tan^{-1}(6.53) = 81.3^\circ\)
Answer: The resultant vector is \(\vec{R} = 96.39\,\text{N}\) at \(81.3^\circ\) above the x-axis.

Equilibrant Vector

  • Equilibrant vector → vector which cancels out the effect of two or more vectors on a body
  • Equilibrant → one vector that cancels the effect that the two or more vectors actually have on a body
  • \(\vec{R}_e = -\vec{R}\)
  • Components of equilibrant vector for this example:
\(R_{ex} = -14.6\,\text{N},\quad R_{ey} = -95.28\,\text{N}\)
\(\vec{R}_e = 96.39\,\text{N}\) at \(81.3^\circ\) below the negative x-axis
  • \(\vec{R}_e + \vec{R} = 0\)
  • If \(\vec{R}\) is a force vector → \(\vec{R}_e\) will be the reaction force that brings the body to equilibrium
  • Equilibrium of a body → maintained if the sum of all external forces acting on it is zero
\(\sum \vec{F} = 0\)

Multiplication of Vectors
Scalar (Dot) Product and Vector (Cross) Product
  • Multiplying a vector by a scalar → changes magnitude or direction (or both)
  • Multiplying one vector by another vector → two common definitions: scalar product and vector product
  • Scalar product of two vectors → just a number (scalar)
  • Vector product of two vectors → itself a vector

Scalar (Dot) Product

  • Denoted by \(\vec{A} \cdot \vec{B}\) → given by:
\(\vec{A} \cdot \vec{B} = AB \cos \theta\)
  • \(\theta\) → angle between vectors \(\vec{A}\) and \(\vec{B}\)
  • Dot product = zero when \(\theta = 90^\circ\) (perpendicular vectors)
  • If two vectors are perpendicular → their scalar product is zero, and vice versa
\(\hat{i} \cdot \hat{j} = \hat{i} \cdot \hat{k} = \hat{k} \cdot \hat{j} = 0\)
\(\hat{i} \cdot \hat{i} = \hat{j} \cdot \hat{j} = \hat{k} \cdot \hat{k} = 1\)
  • In component form (three dimensions):
\(\vec{A} \cdot \vec{B} = A_x B_x + A_y B_y + A_z B_z\)
  • Cosine of angle \(\theta\) between two vectors:
\(\cos \theta = \dfrac{A_x B_x + A_y B_y + A_z B_z}{AB}\)
  • Scalar product is commutative and distributive over addition/subtraction:
\(\vec{A} \cdot \vec{B} = \vec{B} \cdot \vec{A}\)
\(\vec{A} \cdot (\vec{B} \pm \vec{C}) = \vec{A} \cdot \vec{B} \pm \vec{A} \cdot \vec{C}\)
★ Example 1: Find the scalar product of \(\vec{A} = \hat{i} + 2\hat{j} + 3\hat{k}\) and \(\vec{B} = 2\hat{i} + 3\hat{j} + 4\hat{k}\).
\(\vec{A} \cdot \vec{B} = A_x B_x + A_y B_y + A_z B_z\)
\(= (1)(2) + (2)(3) + (3)(4) = 2 + 6 + 12 = 20\)
★ Example 2: Find the value of \(a\) so that \(\vec{A} \cdot \vec{B} = 0\) for \(\vec{A} = 2\hat{i} + a\hat{j} + 3\hat{k}\) and \(\vec{B} = -3\hat{i} - 2\hat{j} + 4\hat{k}\).
\(\vec{A} \cdot \vec{B} = (2)(-3) + (a)(-2) + (3)(4) = -6 - 2a + 12 = 6 - 2a\)
\(0 = 6 - 2a \quad \Rightarrow \quad 2a = 6 \quad \Rightarrow \quad a = 3\)

Vector (Cross) Product
Definition, Properties, and Component Form
  • Vector product of \(\vec{A}\) and \(\vec{B}\) → third vector \(\vec{C}\)
  • \(\vec{C}\) is perpendicular to both \(\vec{A}\) and \(\vec{B}\) and to the plane containing \(\vec{A}\) and \(\vec{B}\)
  • Magnitude of vector product:
\(|\vec{A} \times \vec{B}| = AB \sin \theta\)
  • Product is maximum when \(\theta = 90^\circ\)
  • Product is zero when \(\theta = 0^\circ\) (parallel vectors)
  • If two vectors are parallel → their cross product is zero, and vice versa
\(\hat{i} \times \hat{j} = \hat{k},\quad \hat{i} \times \hat{k} = -\hat{j},\quad \hat{j} \times \hat{k} = \hat{i}\)
\(\hat{i} \times \hat{i} = \hat{j} \times \hat{j} = \hat{k} \times \hat{k} = 0\)
  • Vector product is not commutative but is distributive over addition/subtraction:
\(\vec{A} \times \vec{B} = -\vec{B} \times \vec{A}\)
\(\vec{A} \times (\vec{B} \pm \vec{C}) = \vec{A} \times \vec{B} \pm \vec{A} \times \vec{C}\)
  • In component form → cross product determined using determinant of matrix formed by the two vectors:
\(\vec{A} \times \vec{B} = \det \begin{vmatrix} \hat{i} & \hat{j} & \hat{k} \\ A_x & A_y & A_z \\ B_x & B_y & B_z \end{vmatrix}\)
  • Expanded form:
\(\vec{A} \times \vec{B} = (A_y B_z - A_z B_y)\hat{i} - (A_x B_z - A_z B_x)\hat{j} + (A_x B_y - A_y B_x)\hat{k}\)
★ Example 1: Find \(\vec{A} \times \vec{B}\) for \(\vec{A} = \hat{i} + 2\hat{j} + 3\hat{k}\) and \(\vec{B} = 2\hat{i} + 3\hat{j} + 4\hat{k}\).
\(\vec{C} = (2 \times 4 - 3 \times 3)\hat{i} - (1 \times 4 - 3 \times 2)\hat{j} + (1 \times 3 - 2 \times 2)\hat{k}\)
\(\vec{C} = (8 - 9)\hat{i} - (4 - 6)\hat{j} + (3 - 4)\hat{k} = -\hat{i} + 2\hat{j} - \hat{k}\)
★ Example 2: Find \(\vec{a} \times \vec{b}\) if \(\vec{a} = 3\hat{i} - 2\hat{j} + 5\hat{k}\) and \(\vec{b} = 7\hat{i} + 4\hat{j} - 8\hat{k}\).
\(\vec{c} = ((-2)(-8) - (5)(4))\hat{i} - ((3)(-8) - (5)(7))\hat{j} + ((3)(4) - (-2)(7))\hat{k}\)
\(\vec{c} = (16 - 20)\hat{i} - (-24 - 35)\hat{j} + (12 + 14)\hat{k} = -4\hat{i} + 59\hat{j} + 26\hat{k}\)
★ Example 3: If \(\vec{a} = 8\hat{i} + \hat{j} - 2\hat{k}\) and \(\vec{b} = 5\hat{i} - 3\hat{j} + \hat{k}\), show that \(\vec{a} \times \vec{b} = -5\hat{i} - 18\hat{j} - 29\hat{k}\).
\(\vec{a} \times \vec{b} = \det \begin{vmatrix} \hat{i} & \hat{j} & \hat{k} \\ 8 & 1 & -2 \\ 5 & -3 & 1 \end{vmatrix}\)
\(= (1 \times 1 - (-2) \times (-3))\hat{i} - (8 \times 1 - (-2) \times 5)\hat{j} + (8 \times (-3) - 1 \times 5)\hat{k}\)
\(= (1 - 6)\hat{i} - (8 + 10)\hat{j} + (-24 - 5)\hat{k} = -5\hat{i} - 18\hat{j} - 29\hat{k}\)

Types of Vectors
Null, Position, Coplanar, Collinear, and Unit Vectors
  • Null vectors → vectors of zero magnitude
  • Position vectors → vectors used to locate or define the location of a point or a body in space
\(\vec{r} = (x, y, z) = x\hat{i} + y\hat{j} + z\hat{k}\)
  • Coplanar vectors → vectors which lie in the same two-dimensional plane (may include vectors at different angles to each other)
  • Collinear vectors → vectors which are parallel to each other and act along the same line (can be in same direction or opposite directions)
  • Unit vectors → vectors with length equal to one unit
  • Three unit vectors used to specify direction:
\(\hat{i}\) → 1 unit in x-direction
\(\hat{j}\) → 1 unit in y-direction
\(\hat{k}\) → 1 unit in z-direction
  • Any vector \(\vec{A}\) expressed in terms of magnitude \(A\) and unit vector \(\hat{u}\) in its direction:
\(\vec{A} = A\hat{u}\)
  • Unit vector in direction of given vector \(\vec{A}\) → defined by dividing the vector by its magnitude:
\(\hat{u} = \dfrac{\vec{A}}{A}\)
★ Example 1: Given \(\vec{A} = -2\hat{i} + 1\hat{j} + 3\hat{k}\) and \(\vec{B} = 4\hat{i} + 1\hat{j} - 2\hat{k}\), find:
a) \(\vec{A} - \vec{B}\)    b) \(\vec{B} - 2\vec{A}\)    c) \(\vec{A} + \vec{B}\)    d) unit vector in direction of \(\vec{A} + \vec{B}\)
a) \(\vec{A} - \vec{B}\)
\(\vec{A} - \vec{B} = (-2\hat{i} + \hat{j} + 3\hat{k}) - (4\hat{i} + \hat{j} - 2\hat{k}) = -6\hat{i} + 5\hat{k}\)
b) \(\vec{B} - 2\vec{A}\)
\(\vec{B} - 2\vec{A} = (4\hat{i} + \hat{j} - 2\hat{k}) - 2(-2\hat{i} + \hat{j} + 3\hat{k})\)
\(= 4\hat{i} + \hat{j} - 2\hat{k} + 4\hat{i} - 2\hat{j} - 6\hat{k} = 8\hat{i} - \hat{j} - 8\hat{k}\)
c) \(\vec{A} + \vec{B}\)
\(\vec{A} + \vec{B} = (-2\hat{i} + \hat{j} + 3\hat{k}) + (4\hat{i} + \hat{j} - 2\hat{k}) = 2\hat{i} + 2\hat{j} + \hat{k}\)
  • d) Unit vector \(\hat{U}\) in direction of \(\vec{A} + \vec{B}\) → divide vector by its magnitude
\(|\vec{A} + \vec{B}| = \sqrt{2^2 + 2^2 + 1^2} = \sqrt{4 + 4 + 1} = \sqrt{9} = 3\)
\(\hat{U} = \dfrac{2\hat{i} + 2\hat{j} + \hat{k}}{3} = \dfrac{2}{3}\hat{i} + \dfrac{2}{3}\hat{j} + \dfrac{1}{3}\hat{k}\)
Answer: a) \(-6\hat{i} + 5\hat{k}\)   b) \(8\hat{i} - \hat{j} - 8\hat{k}\)   c) \(2\hat{i} + 2\hat{j} + \hat{k}\)   d) \(\frac{2}{3}\hat{i} + \frac{2}{3}\hat{j} + \frac{1}{3}\hat{k}\)
